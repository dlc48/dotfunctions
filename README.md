README.md

this directory contains the R code of the so called 'dot functions'
- functions are stored under /code/fun/
- the script code/source.r allows to source all R scripts of the folder code/fun/.
  code/source.r requires the string 'path.dotfunctions' defining the path to 
  the git_dotfunctions directory
- ignore the '.DS'-files (OSX stuff)
- ignore the '.r~'-files (backup stuff)
